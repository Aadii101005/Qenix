# Task Manager Application

This is a simple Full Stack Task Manager application 

## Tech Stack
Frontend: Next.js, TailwindCSS, Lucide React, Axios
Backend: NestJS, Mongoose, Passport JWT
Database: MongoDB

## Getting Started

### Prerequisites
Node.js (v18+)
MongoDB (Running locally or on Atlas)

### Setup Backend
1. Go to `backend/`
2. Create `.env` from `.env.example`
3. Run `npm install`
4. Run `npm run start:dev`

### Setup Frontend
1. Go to `frontend/`
2. Run `npm install`
3. Run `npm run dev`

### Credentials
Register a new account on the UI.
Login with your registered email and password.
